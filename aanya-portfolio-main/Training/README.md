# Learn Web Development with AI — Complete Training Package

A two-part, hands-on training program delivered in **Cursor**, taking absolute beginners from "I've never written code" to "I've deployed a live full-stack website with a cloud database."

## Course Structure

| Part | Duration | Focus | Outcome |
|------|----------|-------|---------|
| **Part 1** | 3 hours | Front-End fundamentals + AI prompting for UI/UX | A 4-page responsive UX Designer portfolio running locally |
| **Part 2** | 3 hours | Back-End + Database + Version Control + Deployment | Same portfolio, now full-stack with Supabase, on GitHub Pages |

## The Unified Running Example

Throughout **both** parts we build **one** continuous project:

> **"Aanya Sharma — Senior UX Designer Portfolio"**

- Part 1 builds the static site (Home, About, Projects, Contact).
- Part 2 wires the Contact form to a Supabase database, builds an Admin Dashboard, pushes to GitHub, and deploys live.
- A bonus mini-project (Timer App) demonstrates rapid prototyping at the end of Part 2.

This continuity is intentional — students see the **same project grow** from a static HTML file into a deployed full-stack web application, which is the single most motivating experience for beginners.

## How to Use

**Trainer:** Open `training-guide.md` for the full narrative, teaching notes, prompting strategy callouts, and Q&A breakpoints. Project `slides.html` on screen during delivery.

**Students:** Keep `prompts.md` open in a second window — every prompt is numbered and ready to copy into Cursor.

## Pedagogical Principles Used

1. **One project, two parts** — eliminates context-switching, maximises retention.
2. **Prompt-first teaching** — every concept is introduced *through* a prompt, not before it.
3. **Strategy callouts** — after each prompt the doc explains *why* it was written that way (persona, specificity, consolidation, iteration, demand-explanation).
4. **Built-in Q&A pockets** — five dedicated pause points per part, not rushed.
5. **Copy-paste safe** — every prompt is fenced, self-contained, and ordered.
6. **Beginner-to-advanced ramp** — Part 1 ends where most "intro" courses stop; Part 2 takes them where most "intro" courses never go.
